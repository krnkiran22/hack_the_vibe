# Hack the Vibe - 3D Assets

This directory contains the 3D models and textures used in Hack the Vibe.

## Models
- `tree_74f4dbc9-5e61-4a68-8510-36a3aa70a3a0`: Custom blended tree models for environment variety.
- `rpm.fbx`: Character base models.

## Textures
- Environment maps and skyboxes are located in the `public/textures` directory.

## Usage
Models are loaded via `useGLTF` in the components within `src/components/`.
