# Contributing to Hack the Vibe

Thank you for your interest in contributing to Hack the Vibe! We welcome contributions from the community to help make this AI-powered shooter even better.

## How to Contribute

1.  **Fork the Repository**: Create your own fork of the project.
2.  **Clone the Fork**: Clone your fork to your local machine.
3.  **Create a Branch**: Create a new branch for your feature or bug fix.
    ```bash
    git checkout -b feature/your-feature-name
    ```
4.  **Make Changes**: Implement your changes and ensure they follow the project's style.
5.  **Commit Changes**: Commit your changes with a clear and descriptive message.
6.  **Push to GitHub**: Push your branch to your fork.
    ```bash
    git push origin feature/your-feature-name
    ```
7.  **Submit a Pull Request**: Create a PR from your fork's branch to the main repository's `main` branch.

## Reporting Issues

If you find a bug or have a feature request, please open an issue on the GitHub repository. Provide as much detail as possible, including steps to reproduce the bug.

See the `README.md` for instructions on setting up the frontend, backend, and smart contract development environments.
